# Kayakalp Shopify Theme

This repository contains the Shopify theme for the Kayakalp store.

## Structure

- `assets/` — CSS and JavaScript assets
- `layout/` — theme layout templates
- `sections/` — reusable Liquid sections
- `snippets/` — snippet templates
- `templates/` — page and collection templates
- `config/` — theme settings and schema
- `locales/` — theme translations

## Development

Use the Shopify CLI to preview and deploy the theme:

```bash
shopify theme serve
```

To push the theme to the store in unpublished mode:

```bash
shopify theme push --store kayakalp-3.myshopify.com --unpublished
```
